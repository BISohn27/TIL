# EL

- **${값}** 

  - <%= 값 %>
  - <% out.print(값); %> 
  - 문자열(" ") , 문자열(' '), 정수값, 부동소수점, 불리언 사용 가능

- **${객체 값}**

  - 객체에 속한 값도 위의 표현식을 통해 값을 불러올 수 있다.

  - 객체 값은 해당 객체를 setattribute 할 때 입력한 키 값

  - 객체=request.getAttribute("key"); 로 값을 request로부터 꺼내서

    <%= 객체 값 %> 로 불러온 값을 출력하는 것을 생략한 태그

  - **${배열이름[n]}** : 배열에서 n번째 값을 가져옴

  - **${리스트키값[n]}** : 리스트가 저장되어 있는 키값에서 n번째 값을 가져옴

  - **${map이름.map키}** : 맵에서 해당 키에 대한 값을 가져옴

    - **${객체이름.객체필드}** : 객체 이름으로 객체 필드를 반환하는 getter 함수를 호출하여 										값을 얻어올 수 있다.
