# HTML

- 웹을 이루는 가장 기본적인 building block
- web contents의 구조와 의미를 결정
- 웹 브라우저 상에 보여질 수 있게 만들어진 문서
- 구조적인 annotation이 들어가 문장의 구조를 표시하면서 작성하는 문서
- W3C에 의해 정의된 표준에 따라 웹 브라우저가 텍스트로 적혀진 문장들을 구조적으로 변환
- [MDN]([MDN Web Docs (mozilla.org)](https://developer.mozilla.org/ko/)
- 태그 마다 지원되는 브라우저가 정해져 있으니 확인하며 사용하는게 필요.
- [HTML 검사](http://validator.w3.org)
- 웹 브라우저는 각각의 구역이 박스 구조로 이뤄져 있다.

## Tag

- <> </> 표시의 annotation을 통해 웹 브라우저가 해당 문서를 구조적으로 해석하여

  해당 구조를 가진 화면을 클라이언트에게 제공

- **<>contents</>** : annotating된 문장 

- 태그 안에 child 태그가 들어갈 수 있음

- **<!DOCTYPE html>**: doc 타입이 html이라는 표시.

## Head  

- 사용자에게 보여지는 ui적인 요소가 없는 태그
- 구글 등 검색 엔진으로 검색 했을 때 검색 되거나 제목이나 아이콘, css 파일 연결 (Meta Data)

## Body

- 사용자에게 보여지는 가장 최상위 태그
- 정의되어 있지 않은 태그를 사용해도 브라우저 엔진이 해당 에러를 최대한 처리를 함.
